宠物健康 AI 智能诊断系统

基于 Qwen2.5-VL 多模态大模型与 RAG 技术的全科宠物医疗辅助平台

微信小程序 全科诊断 异宠支持 行为分析

🏗️ 系统架构 (System Architecture)

架构设计说明：

系统采用了“端云协同 + 后融合”的设计模式。前端小程序负责采集多模态数据（图片、音频、文本），后端 FastAPI 网关分发任务：视觉流进入 Qwen-VL 进行病灶识别，音频流进入声学模型提取 MFCC 特征进行情绪分类。最终由 LLM 结合 Milvus 知识库（RAG）生成综合诊断报告。

🩺 AI 临床诊断模块 (Clinical Diagnosis)

基于 CV 计算机视觉与医学知识库，覆盖四大核心科室。

🦠

皮肤病变检测

通过对比学习区分真菌感染（猫藓）、螨虫、细菌性皮炎与过敏。支持患处分割与严重程度分级。

Contrastive Learning Few-shot
👂

耳道健康分析

基于耳镜图像识别耳螨、马拉色菌感染（褐色分泌物）及中耳炎。结合颜色与纹理特征提取。

Texture Analysis Color Histogram
👁️

眼科疾病筛查

检测角膜混浊、樱桃眼、结膜炎及白内障。精准定位眼部区域，排除泪痕干扰。

YOLOv8-Seg ROI Extraction
💩

消化系统分析

排泄物：基于 Bristol 分级标准自动评分（Grade 1-5），检测寄生虫与潜血。
呕吐物：分析颜色（胆汁/血液）与未消化食物，预警细小病毒等急症。

Feature Classification Medical Logic

🧠 行为与认知分析 (Behavior & Cognition)

😺

多模态情绪识别

融合视觉（耳位/瞳孔/尾巴状态）与听觉（叫声频率/音高），识别焦虑、疼痛、兴奋等 6 种情绪状态。

Multimodal Fusion Audio-Visual
🐕

异常行为监控

识别睡眠、玩耍、跛行等行为模式。结合时间序列数据，预警骨关节炎或分离焦虑倾向。

Action Recognition Time-Series
🔎

精准品种鉴定

支持 100+ 种犬猫品种识别。基于细粒度图像分类技术，辅助兽医判断品种特异性遗传病风险。

Fine-grained Classification

🐹 异宠专科支持 (Exotic Pets)

🦎

爬行类与小型哺乳动物支持

针对非犬猫类宠物建立专门的知识库与识别模型，填补市场空白。

  • 支持物种：仓鼠、兔子、龙猫、乌龟、蜥蜴、鹦鹉等。
  • 特异性诊断：
    🐀 啮齿类：湿尾症、牙齿过长、脱毛检测。
    🐢 爬行类：龟壳腐甲病、软壳病、蜥蜴代谢性骨病识别。
    🐦 鸟类：羽毛啄拔、鸟喙生长异常检测。
